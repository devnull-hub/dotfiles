# Sync Dotfiles and Config Using Git

# Requirements
* Git
* Curl

# Install

Install config tracking in your `$HOME` by running:

`curl -Lks https://raw.githubusercontent.com/devnull-hub/dotfiles/master/.bin/cfg-init.sh | /bin/bash`
